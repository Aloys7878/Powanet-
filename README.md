POWANET v2.7 Full Build (Demo-ready)

How to use:
1. unzip, run `npm install` then `npm run dev` for local testing.
2. Upload the ZIP to Pi App Studio using "Upload code" (App Studio supports code upload).
3. In Pi App Studio edit app: replace app with uploaded code, then publish to Testnet.

This bundle contains:
- pages/ (index, api stubs)
- components/ (EgoChat, NetworkGrid, Wallet)
- public/assets (SVG placeholders)
- styles/globals.css
